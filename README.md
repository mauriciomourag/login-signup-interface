<h1 align="center">Authentication Login & Signup Interface</h1>

<p align="center">
Interface de autenticação com telas de <strong>Login e Cadastro</strong>, desenvolvida para prática de conceitos de desenvolvimento frontend.
</p>

<hr>

<h2>📘 Sobre o Projeto</h2>

<p>
Este projeto implementa uma interface de autenticação contendo telas de 
<strong>login e cadastro de usuário</strong>. 

O objetivo principal foi aplicar na prática conceitos aprendidos durante os estudos de desenvolvimento web, incluindo:

<ul>
<li>Estruturação de formulários</li>
<li>Componentização de interface</li>
<li>Manipulação de estados e inputs</li>
<li>Design de telas de autenticação</li>
</ul>

O projeto serve como exercício de implementação de um fluxo básico de autenticação em aplicações web modernas.
</p>

<hr>

<h2>▶️ Como Executar o Projeto</h2>

<h3>1️⃣ Clonar o Repositório</h3>

<pre>
git clone https://github.com/mauriciomourag/auth-login-signup-ui.git
</pre>

<h3>2️⃣ Acessar a Pasta</h3>

<pre>
cd auth-login-signup-ui
</pre>

<h3>3️⃣ Instalar Dependências</h3>

<pre>
npm install
</pre>

<h3>4️⃣ Iniciar o Projeto</h3>

<pre>
npm start
</pre>

<p>
Após iniciar o servidor de desenvolvimento, a aplicação estará disponível em:
</p>

<pre>
http://localhost:3000
</pre>
